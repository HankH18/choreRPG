# RPG Style Chore Counter

App features:
 * Click a chore and it's added to the tally


Technologies used:

 * Front-End
 	* React with Flux
 	* Backbone Routing


### Installation
```
git clone git@github.com:hankh18/choreRPG
cd choreRPG
npm install
npm run go

go to `localhost:3000` in your browser
```
